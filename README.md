## How we built it
We built the NicheCard website primarily using Javascript and React. Our NicheAI Analyzer is built using Node.js and utilizes ChatGPT 5 nano LLM to make card comparison breakdowns for the user. We also utilized an opensource api to collect data on 170+ of the worlds most popular credit cards.
